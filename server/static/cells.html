Cells & predictions — the pre-registered matrix

Once disposition became the question, the program was hardened as a pre-registered experimental matrix. Every hypothesis was written down, in falsifiable form, before the test that could kill it ran. This page defines the matrix, shows which cells executed and what each returned with data, and lists every prediction. The narrative route through these cells is on the Routes page.

Cell
One unit of the paper-hardening matrix: a specific, isolated test with defined arms, endpoints, and a success/failure criterion fixed in advance. Cells were numbered 1–10 on 2026-07-11, with a dose-response cell added after Cell 2 flagged it as the priority open question.
Prediction (P#)
A falsifiable claim registered before its cell ran, each paired with an explicit "falsified if…" condition. Registering the prediction in advance is what makes a refutation meaningful rather than a post-hoc story. Hover any P# chip to read the full prediction.
Endpoints
The three numbers every installation cell reports: seat density (behavior emitted at the specialist's own output), final CDS (behavior surviving at the pipeline's synthesized output), and the case-7 gate (behavior emitted on the trigger-light control, where the correct amount is zero).
Arms
A′ conversion-control baseline · spec behavior-spec prompting · SFT fine-tune on chosen exemplars · ORPO reference-free preference optimization.

Cell 1 SFT-on-chosen control P1 ✓ Confirmed

The gatekeeper — the only cell that could invalidate the headline. Identical LoRA config, iterations, and seed as ORPO; only the objective differs (SFT on chosen responses vs. ORPO's chosen-vs-rejected preference). Single run/case.

EndpointA′ baselineSFT-on-chosenORPO

Case-7 gate—1.59 ✗ failed0.00 ✓

SFT bought the biggest raw seat lift, then deployed it indiscriminately (hedged where nothing warranted it) and fragilely (worst synthesis survival). The contrastive signal — SFT's only missing ingredient — is load-bearing for both quality properties.

Cell 2 Variance pass — 140 runs, bootstrap 95% CIs ✓ Executed

The rigor cell. It revised Cell 1's single-run numbers: ORPO's flashy single-run "1.2× lift / best-CDS" turned out to be noise. 5 seeds × 4 arms × 7 cases.

Survives error bars: prompting + SFT install real ~2× seat lifts (CIs clear of baseline) that synthesis strips (final CDS at/below baseline); ORPO installs no magnitude but uniquely suppresses unwarranted hedging (0.15, below baseline's 0.96). This cell flagged dose-response as the priority open question.

Cell 6 Synthesizer-register ablation — 72 runs P6 ✓ Confirmed, own prediction overturned

The decisive cell. 3 Leads × PRESERVE on/off × base vs. hot (SFT) seat. Values are final-output density; seat input density in parentheses.

LeadPRESERVE, basePRESERVE, hot seatno-PRESERVE, baseno-PRESERVE, hot
Phi-41.17 (0.78)0.61 (1.42) — inverts0.52 (0.54)0.31 (2.15)
gpt-oss0.63 (0.87)0.86 (1.81)0.20 (0.86)0.13 (2.33)
Qwen2.51.24 (0.79)1.06 (2.41)0.24 (0.92)0.62 (1.93)

Three findings: (1) registers are writer-specific (each Lead has its own band, Qwen hottest, gpt-oss coolest); (2) over-dense input does not transmit and can invert output (Phi-4: 1.17 → 0.61); (3) PRESERVE is a gain control — removing it collapses density 2–5× on every Lead, overturning our own registered prediction that instructions wouldn't matter. Mechanism: final disposition ≈ f(register × instructions), nearly independent of seat input.

+dose Dose-response — 35 runs, 292 pairs (3.2×) ○ Dose-invariant null

Added after Cell 2. Extended pair generation to 500 prompts → 292 gated pairs, retrained epoch-matched to the 91-pair run (1,056 iterations) so the only variable is the number of distinct examples.

ArmSeat density [CI]Final CDS [CI]Case-7 gate
A′ baseline0.84 [0.63, 1.06]0.85 [0.63, 1.07]0.96

Tripling the data moved seat magnitude by zero (0.84 vs 0.85 vs 0.84), even as training preference-accuracy rose 0.48 → 0.94. Magnitude is bounded by the register, not by data — the corollary Cell 6 predicts. (Honest wrinkle: the suppression itself weakened slightly with dose, gate 0.15 → 0.49, still below baseline.)

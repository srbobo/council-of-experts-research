Cells & predictions — the pre-registered matrix

Once disposition became the question, the program was hardened as a pre-registered experimental matrix. Every hypothesis was written down, in falsifiable form, before the test that could kill it ran. This page defines the matrix, shows which cells executed and what each returned with data, and lists every prediction. The narrative route through these cells is on the Routes page.

Cell
One unit of the paper-hardening matrix: a specific, isolated test with defined arms, endpoints, and a success/failure criterion fixed in advance. Cells were numbered 1–10 on 2026-07-11, with a dose-response cell added after Cell 2 flagged it as the priority open question.
Prediction (P#)
A falsifiable claim registered before its cell ran, each paired with an explicit "falsified if…" condition. Registering the prediction in advance is what makes a refutation meaningful rather than a post-hoc story. Hover any P# chip to read the full prediction.
Endpoints
The three numbers every installation cell reports: seat density (behavior emitted at the specialist's own output), final CDS (behavior surviving at the pipeline's synthesized output), and the case-7 gate (behavior emitted on the trigger-light control, where the correct amount is zero).
Density · CDS
Density is the raw rate of disposition: the number of the five behavior-family markers found in a piece of text, per 1,000 characters. "Seat density" measures it on the specialist's own turn; measured on the final answer it feeds the CDS. CDS (Composite Disposition Score) = density × √(distinct families ÷ 5) — the square root softly penalizes narrow emission (one behavior repeated loudly) without letting a single absent family zero the score. Both are per-character, so they reward concision; a judge-validated, per-claim replication is planned.
Arms
A′ conversion-control baseline · spec behavior-spec prompting · SFT fine-tune on chosen exemplars · ORPO reference-free preference optimization.
Dose · dose-response
Borrowed from pharmacology: dose is the amount of preference-training data — the number of content-controlled pairs the seat is trained on (91 in the original run, 292 in the dose-response cell, hence "3.2× dose"). A dose-response test increases the dose and asks whether the installed behavior scales with it, exactly as one would vary a drug dose and measure the response. To keep dose the only variable, the higher-dose retrain is epoch-matched — the same number of passes through the data (16 epochs) — so more pairs, not longer training, is what changes. The finding was a dose-invariant null: tripling the dose moved the response by zero.

Cell 1 SFT-on-chosen control P1 ✓ Confirmed
Hypothesis (P1)

SFT-on-chosen installs weaker, less-responsive disposition than ORPO — the contrastive preference signal is load-bearing, not mere exemplar exposure. (Falsified if SFT matches ORPO on seat density AND case-7 gate AND synthesis durability.)

Experiment

The gatekeeper — the only cell that could invalidate the headline. Train an SFT arm on the same pairs' chosen responses with identical LoRA config, iterations, and seed as ORPO, so only the objective differs. Bench all three arms (A′ / SFT / ORPO) on the three endpoints. Single run/case.

Results
EndpointA′ baselineSFT-on-chosenORPO
Seat density0.821.69 (2.1×)1.00 (1.2×)
Case-7 gate—1.59 ✗ failed0.00 ✓
Final CDS0.6240.443 (worst)0.738 (best)

SFT bought the biggest raw seat lift, then deployed it indiscriminately (hedged where nothing warranted it) and fragilely (worst synthesis survival). The contrastive signal — SFT's only missing ingredient — is load-bearing for both quality properties. P1 confirmed.

Cell 2 Variance pass — 140 runs, bootstrap 95% CIs ✓ Executed
Hypothesis

Cell 1's single-run verdicts are trustworthy only if they survive replication. Do the installed lifts, the gate failures, and the stripping hold up with error bars — or were any of them noise at n=1?

Experiment

Run 5 seeds of each arm (A′ / spec / SFT / ORPO) across all 7 cases — 140 runs — and report each endpoint as a mean with a bootstrap 95% CI (104 resamples over run-level observations).

Revised Cell 1. ORPO's flashy single-run "1.2× lift / best-CDS" was noise — at n=5 it sits on baseline. What survives the error bars: prompting + SFT install real ~2× seat lifts (CIs clear of baseline) that synthesis strips (final CDS at/below baseline); ORPO installs no magnitude but uniquely suppresses unwarranted hedging (0.15, below baseline's 0.96). This cell flagged dose-response as the priority open question.

Cell 6 Synthesizer-register ablation — 72 runs P6 ✓ Confirmed, own prediction overturned
Hypothesis (P6)

Synthesis durability holds across the 3 local Leads (Phi-4, gpt-oss-20B, Qwen2.5-7B) and weakens without PRESERVE instructions. Following Cell 10's refutation, the sharper claim under test: each Lead writes at its own register (a characteristic output-density band), largely independent of seat input — so "stripping" is just an over-dense input being normalized to the Lead's band. (Falsified if durability is Phi-4-only, or if PRESERVE makes no difference.)

Experiment

A full factorial: 3 Leads × PRESERVE on/off × base vs. hot (SFT) seat × 6 cases = 72 runs. Values below are final-output density; seat input density in parentheses, so you can read whether input transmits.

Results
LeadPRESERVE, basePRESERVE, hot seatno-PRESERVE, baseno-PRESERVE, hot
Phi-41.17 (0.78)0.61 (1.42) — inverts0.52 (0.54)0.31 (2.15)
gpt-oss0.63 (0.87)0.86 (1.81)0.20 (0.86)0.13 (2.33)
Qwen2.51.24 (0.79)1.06 (2.41)0.24 (0.92)0.62 (1.93)

Three findings: (1) registers are writer-specific (each Lead has its own band, Qwen hottest, gpt-oss coolest); (2) over-dense input does not transmit and can invert output (Phi-4: 1.17 → 0.61); (3) PRESERVE is a gain control — removing it collapses density 2–5× on every Lead, overturning our own registered prediction that instructions wouldn't matter. Mechanism located: final disposition ≈ f(register × instructions), nearly independent of seat input.

Cell 10 Marker-position analysis — 102 runs P8 ✗ Falsified
Hypothesis (P8)

Trained behaviors are content-entangled — markers co-occur with entities/statutes in-sentence — while prompted behaviors are detached boilerplate. Entangled markers should therefore survive synthesis at higher rates regardless of which arm produced them. (Falsified if survival is arm-dependent but not position-dependent.)

Experiment

No new runs — a post-hoc analysis of 102 existing audited runs. For each marker, classify it as entangled (co-occurs with content signals: digits, §, U.S.C., case cites, proper nouns) or detached, then measure each arm's entangled share against its retention (final-output density ÷ seat density) and correlate the two.

Results

Falsified. Survival is arm-dependent, not position-dependent — SFT's markers are the most content-woven yet retain worst; correlation(entanglement, retention) r = −0.10 ≈ 0. Entanglement does not explain stripping. The register (Cell 6) does — and this refutation is what designed Cell 6.

+dose Dose-response — 35 runs, 292 pairs (3.2×) ○ Dose-invariant null
Hypothesis

The natural objection to Cell 2's ORPO magnitude-null: the seat simply wasn't trained on enough pairs. More preference data should install seat-level magnitude where 91 pairs could not.

Experiment

Extend pair generation to 500 prompts → 292 gated pairs (3.2× dose) and retrain epoch-matched to the 91-pair run (1,056 iterations, 16 epochs) so exposure-per-example is held constant and the only variable is the number of distinct examples. Bench 35 runs; compare against baseline and the 91-pair arm.

Results
ArmSeat density [CI]Final CDS [CI]Case-7 gate
A′ baseline0.84 [0.63, 1.06]0.85 [0.63, 1.07]0.96
ORPO, 91 pairs0.85 [0.58, 1.15]0.66 [0.50, 0.85]0.15
ORPO, 292 pairs (3.2×)0.84 [0.58, 1.11]0.69 [0.51, 0.90]0.49

Null — the objection fails. Tripling the data moved seat magnitude by zero (0.84 vs 0.85 vs 0.84), even as training preference-accuracy rose 0.48 → 0.94: the model learned the preference far better and emitted no more behavior for it. Magnitude is bounded by the register, not by data — the corollary Cell 6 predicts. (Honest wrinkle: the suppression itself weakened slightly with dose, gate 0.15 → 0.49, still below baseline.)

End result · the thesis
The Last Writer Wins
Pipeline disposition is not additive across stages — it is set by the last writer's register and that writer's instructions. Prompting and SFT install loud, indiscriminate behavior the aggregator strips; preference optimization installs quiet, responsive behavior that survives by conformity, and no amount of data raises it. Tune the synthesizer; evaluate at the pipeline's mouth, not the seat.
